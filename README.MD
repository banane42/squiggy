### Postgres commands

connect  
`psql {database name}`

clear  
`\! clear`

display tables  
`\d`

describe table  
`\d+ {table name}`

## Python venv commands
create venv  
`python3 -m venv .venv`

activate venv  
`source .venve/bin/activate`

install requirements  
`python3 -m pip install -r requirements.txt`

run flask app  
`flask --app src/app.py run`

deactivate venve  
`deactivate`
